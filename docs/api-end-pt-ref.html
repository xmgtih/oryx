Bundled Serving Layer Apps

Oryx bundles several end-to-end applications, including a Serving Layer with REST endpoints. The bundled app endpoints are:

Collaborative filtering / Recommendation

  • /recommend
  • /recommendToMany
  • /recommendToAnonymous
  • /similarity
  • /similarityToItem
  • /knownItems
  • /estimate
  • /estimateForAnonymous
  • /because
  • /mostSurprising
  • /popularRepresentativeItems
  • /mostPopularItems
  • /mostActiveUsers
  • /item/allIDs
  • /ready
  • /pref
  • /ingest

Classification / Regression

  • /predict
  • /classificationDistribution
  • /train

Clustering

  • /assign
  • /distanceToNearest
  • /add
